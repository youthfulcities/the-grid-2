Sur cette page, vous trouverez toutes les villes traitées par THE GRID.

Cliquez sur les cartes pour trouver les ensembles de données qui contiennent des informations sur cette ville. Les indices sont des ensembles de données fondés sur des valeurs pour des données quantifiables. Par exemple, le salaire minimum a une valeur dans chaque ville que nous pouvons mesurer. Les ensembles de données de l’enquête enregistrent les réponses aux questionnaires. Les ensembles de données de l’entretien contiennent des citations des participant·e·s.
