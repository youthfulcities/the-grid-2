Accueil / Tous les ensembles de données

Sur cette page, vous trouverez tous les ensembles de données de THE GRID.


Cliquez sur les cartes pour voir quelles visualisations sont disponibles pour chaque ensemble de données. La visualisation sous forme de tableau affiche les données dans une feuille de calcul. La visualisation sous forme de carte affiche les points de données sur une carte interactive. La fonctionnalité Analyse vous permet d’établir un graphique des données. Les constats sont des visualisations supplémentaires fournies pour comparer les données et les visualiser dans un format plus convivial. La fonctionnalité Citation affiche les citations directes des participant·e·s à l’enquête ou à l’entretien, ainsi que leurs informations démographiques. Vous trouverez également un lien vers la méthodologie détaillée pour chaque ensemble de données, lorsqu’elle est disponible.


Les indices sont des ensembles de données fondés sur des valeurs pour des données quantifiables. Par exemple, le salaire minimum a une valeur dans chaque ville que nous pouvons mesurer. Les ensembles de données de l’enquête enregistrent les réponses aux questionnaires. Les ensembles de données de l’entretien contiennent des citations des participant·e·s.
